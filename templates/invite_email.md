Here's how to get started:

1. DOWNLOAD DISCORD (if you don't have it)
   Go to https://discord.com/download and install Discord on your phone or computer (phone recommended, as it builds mobile apps).
   Create a free account if you don't have one.

2. JOIN THE SERVER
   Click this invite link: {invite_url}
   (This link expires in 7 days and can only be used once.)

3. START CHATTING WITH THE BOT
   Once you're in the server, find "ClaudeAppBot" in the member list on the right side.
   Click the bot's name > "Message" to start a private chat.

4. BUILD YOUR FIRST APP
   Send this message to the bot:
   /buildapp a pomodoro timer with task categories

   That's it! Wait a few minutes and you'll get a working app.

QUICK COMMAND REFERENCE
  /buildapp <describe anything>   build a full app from a description
  @appname <request>              tell the bot to change your app
  /demo web                       preview your app in the browser
  /testflight                     publish to the iOS App Store for testing
  /playstore                      publish to Google Play Store for testing
  /help                           see all commands

TIPS
  - Be specific! "A workout tracker with sets, reps, and a rest timer"
    works better than "a fitness app"
  - Build the core idea first, then add features one at a time
  - If something breaks, the bot auto-fixes it
